Pretend Client Page

This is a pretend client page for the Secure Citizen Camera Module found at https://github.com/securecitizen/sc-camera-module. The demo page imports a typescript file during development. It comese preconfigured with TailwindCSS.

The typescript file is compiled to javascript, a type definitions file, and a css file when hosted in a GitHub Pages page during production.
